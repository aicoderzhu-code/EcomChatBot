## 🚀 如何使用

### 快速开始

```bash
# 1. 进入测试目录
cd /Users/zhulang/work/ecom-chat-bot/backend/tests

# 2. 安装测试依赖
pip install -r requirements-test.txt

# 3. 运行所有测试
./run_all_tests.sh

# 4. 查看报告
open htmlcov/index.html
```

### 运行特定测试

```bash
# 运行健康检查测试
pytest test_01_health.py -v

# 运行租户管理测试
pytest test_03_tenant.py -v

# 运行支付管理测试
pytest test_07_payment.py -v

# 运行E2E测试
pytest test_e2e.py -v
```

### 按标记运行

```bash
# 冒烟测试
pytest -m smoke -v

# 快速测试
pytest -m fast -v

# 管理员模块测试
pytest -m admin -v

# 支付模块测试
pytest -m payment -v

# E2E测试
pytest -m e2e -v
```
